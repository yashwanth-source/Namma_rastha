Namma Raste Reporter

Overview

Namma Raste Reporter is an Android-based civic issue reporting application developed to help citizens report road-related problems such as potholes, garbage accumulation, water leakage, damaged streetlights, and traffic issues. The application uses modern Android technologies and GenAI-assisted development to provide a smooth and user-friendly experience.

The app allows users to capture images, attach GPS location details, and submit complaints directly through a mobile interface. The system stores reports securely using Firebase and local Room Database support.

System Architecture

1. User opens the application.


2. User captures issue image using camera.


3. GPS location is fetched automatically.


4. Complaint details are entered.


5. Data is stored locally using Room Database.


6. Complaint is uploaded to Firebase cloud server.


7. Authorities can access and review reports.
